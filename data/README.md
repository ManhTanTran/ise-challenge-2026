# Dữ Liệu Local

Folder này dùng để chứa dữ liệu challenge trên máy local.

```text
data/
  sample_data_lake/
    0.Sample_Data.xlsx
    Data-Lake/
```

`sample_data_lake/` được ignore khi push GitHub để tránh commit dữ liệu lớn và cache phát sinh. Khi clone repo mới, hãy đặt lại dữ liệu theo đúng cấu trúc trên.

Các lệnh trong README root giả định dữ liệu nằm ở:

```text
data/sample_data_lake/
```
